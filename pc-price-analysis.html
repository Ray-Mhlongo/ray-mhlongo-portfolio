PC Price Analysis

Advanced marketplace indexing using SQL and Power BI to standardize raw PC hardware data for meaningful price comparison insights.

Project Overview

This project consolidates PC hardware listings from multiple marketplace sources into a standardized analytical dataset. The goal is to improve price transparency by aligning inconsistent naming, specifications, and listing structures.

Problem Statement

Raw listings often include inconsistent product names, duplicate records, and missing specification fields. Without normalization, comparing prices across vendors can produce unreliable conclusions and poor decision support.

Tools Used

  • SQL for extraction, cleaning, and relational modeling
  • Power BI for dashboard design and comparative insights
