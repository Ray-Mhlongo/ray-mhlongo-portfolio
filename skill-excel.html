Spreadsheet model for traffic review

I used the packet CSV as an Excel reporting project. The workbook idea is simple: clean fields, create helper columns, summarize traffic with pivots, and highlight rows that deserve attention.

Project Structure

Business Problem

Raw network exports are hard to read in spreadsheet form without cleanup, formulas, and a clear dashboard view.

Dataset Overview

The workbook uses packet fields such as Source, Destination, Protocol, Length, and Info.

Solution Approach

I designed helper formulas, pivot summaries, conditional formatting, and Power Query style cleanup steps.

Technologies Used

Excel tables, XLOOKUP, INDEX MATCH, SUMIFS, IF statements, pivot tables, charts, and conditional formatting.

Demonstration

Final Outcome

Key Insight

Pivot-style summaries make the dominant protocols easy to compare without reading thousands of rows.

Visualizations

The dashboard layout supports KPI cards, protocol charts, conditional formatting, and quick review statuses.

Business Value

The workbook gives a non-technical reviewer a cleaner way to understand packet activity.
