# GNOSIS Therapy Website

Professional multi-page therapist website for **GNOSIS Therapy and Wellness Service**.

## Pages

- `index.html` - Home
- `about.html` - About
- `services.html` - Services
- `contact.html` - Contact

## Tech

- `style.css` - Shared responsive styling
- `script.js` - Contact form validation and small UI helpers
- `assets/` - Placeholder logo and images

## Deploy to GitHub Pages

1. Push this folder contents to a GitHub repository.
2. In repository settings, enable **GitHub Pages**.
3. Set source to the root branch/folder containing these files.
4. Your site will publish automatically.
