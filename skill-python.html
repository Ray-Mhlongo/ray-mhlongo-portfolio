Pandas workflow for CSV processing

I used Python to show how I would automate the cleanup and summary process for the provided packet capture CSV files.

Back Home

Project Structure

Business Problem

Manual spreadsheet review becomes slow when the file has hundreds of thousands of packet rows.

Dataset Overview

The CSV includes source, destination, protocol, length, and packet information fields.

Solution Approach

I load the CSV, remove incomplete records, group protocol activity, and export clean summaries for reporting.

Technologies Used

Python, Pandas, CSV processing, automation logic, and chart-ready summary outputs.

Code Demo

import pandas as pd

packets = pd.read_csv("Midterm_53_group.csv")
clean = packets.dropna(subset=["Source", "Destination"])
summary = clean.groupby("Protocol")["Length"].agg(["count", "mean"])
print(summary.sort_values("count", ascending=False).head())

Output: TCP and TLSv1.3 dominate the capture, with ICMP useful for connectivity checks.

TCP 314,873 TLSv1.3 71,625 ICMP 2,690

Final Outcome

Key Insight

Python makes the analysis repeatable, which is useful when new packet exports arrive.

Visualizations

The grouped output can feed Matplotlib charts, Excel pivots, or Power BI dashboards.

Business Value

Automation reduces manual cleanup and speeds up the reporting workflow.
