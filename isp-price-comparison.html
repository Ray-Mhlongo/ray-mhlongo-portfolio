Finding better value internet plans

I compared local internet packages using cost per Mbps to show which plans offered stronger value for residents and small businesses.

Back to Projects

Case Study

Problem

Customers often choose internet plans by brand or monthly price without seeing the real value.

What I did

  • Collected public package data from ISP sources.
  • Cleaned prices, speeds, plan names, and connection types.
  • Calculated cost per Mbps.
  • Built a Power BI dashboard for comparison.

Tools used

Excel, Power BI, public data collection, and value analysis.

Key outcome

The analysis showed that a higher price does not always mean better value. Cost per Mbps made the decision clearer.
