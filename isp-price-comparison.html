Local ISP Value-for-Money Analysis – Small Town Case Study (South Africa)

This project evaluates the value-for-money of ISP packages offered to residents and businesses in a small South African town, using objective price and performance metrics to guide better decisions.

Project Overview

This analysis compares ISP packages available in a small South African town to determine whether long-standing ISP choices truly deliver the best value. Using publicly available data, the project evaluates fibre and wireless offerings across download speed, monthly cost, installation cost, and overall cost efficiency. The outcome is a decision-ready view of value-for-money that does not rely on internal business data.

Project Objective

  • Compare ISP packages objectively using cost-per-Mbps
  • Identify the best-value package offered by each ISP
  • Demonstrate how public data can be transformed into actionable insights
  • Showcase practical analytics skills aligned to business decision-making

Methodology: Data Collection

Data was collected manually from official ISP websites and publicly available sources. The dataset focuses on ISPs that actively serve the selected town, avoiding nationwide averages.

  • ISP name
  • Connection type (Fibre or Wireless)
  • Coverage scope (town-level availability)
  • Package name
  • Download speed (Mbps)
  • Monthly price
  • Installation cost
  • Router inclusion
  • Source URL

Methodology: Data Preparation

Data cleaning and preparation were performed in Microsoft Excel.

  • Structured data into relational tables (ISPs and Packages)
  • Standardized package naming for consistency
  • Removed unused or null columns
  • Created calculated fields for analysis

Cost per Mbps = Monthly Price ÷ Download Speed (Mbps)

A “Best Value” flag was created using Excel formulas to identify the lowest cost-per-Mbps package for each ISP.

Methodology: Data Visualization

The cleaned dataset was imported into Power BI to create an interactive dashboard with consistent styling for portfolio presentation.

  • Bar chart: Best-value ISP plans (cost per Mbps)
  • Column chart: Download speed of best-value plans
  • Table: ISP package details for best-value plans

Key Features of the Dashboard

  • Best-value comparison to highlight the most cost-efficient package per ISP
  • Speed vs cost insight showing that higher speeds do not always equal better value
  • ISP-level analysis for direct comparison between providers serving the same town
  • Clean visual design optimized for clarity and portfolio presentation

Outcomes

The analysis shows that objective price-performance comparison can reveal better alternatives than those chosen through habit or brand loyalty. For local businesses, it provides a practical framework to reassess internet expenses and value. For the portfolio, it demonstrates end-to-end analytics skills in data sourcing, Excel-based calculations, and Power BI visualization within a real-world business context.
