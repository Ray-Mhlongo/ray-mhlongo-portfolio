Flagship Project - Value Analysis

ISP Price Comparison

I compared internet packages in a small South African town to show which plans offered better value for money.

View Dashboard Back to Projects

Case Study

Short version: price alone was not enough, so I compared price against speed.

Problem

Customers may choose internet plans based on habit, brand, or monthly price without knowing the real value.

What I Did

  • Collected package details from public ISP sources.
  • Cleaned plan names, prices, speeds, and connection types.
  • Calculated cost per Mbps for fair comparison.
  • Built a dashboard to compare providers side by side.

Tools Used

Excel Power BI Cost Analysis

Key Outcome

The analysis showed that higher speed or higher price does not always mean better value. Cost per Mbps made the comparison clearer.

Key Insights

The dashboard turns public pricing data into a simple decision tool.

Best Value

Cost per Mbps helped identify the strongest package from each provider.

Speed vs Price

The comparison showed where higher speed was worth the extra cost.

Decision Support

The result helps residents and small businesses review internet costs more clearly.
