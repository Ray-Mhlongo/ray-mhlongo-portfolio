Local ISP Price Comparison

Analysis comparing local internet service provider pricing to evaluate cost differences, package offerings, and value for money.

Project Overview

This project compares ISP plans on cost, speed tiers, and package benefits so residents and small businesses can evaluate service options objectively.

Problem Statement

ISP packages are marketed differently, which makes direct comparison difficult. A normalized model is required to evaluate equivalent value across providers.
